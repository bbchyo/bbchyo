## 🛠️ Expertise & Featured Projects

### 📊 Qualitative Data Analysis (QDA)
* **[obsidian-quadro](https://github.com/bbchyo/obsidian-quadro):** An Obsidian Plugin for social-scientific QDA—an open alternative to MAXQDA and atlas.ti using Markdown.
* **[openqda](https://github.com/bbchyo/openqda):** Free open collaborative qualitative data analysis software.
* **[Qualimind QDA](https://github.com/bbchyo/qualimind-qda):** Cutting-edge qualitative data analysis tool integrating powerful AI capabilities.

### 🧠 AI in Mental Health & NLP
* **[suicidal-text-detection](https://github.com/bbchyo/suicidal-text-detection):** Building a suicidal text detection model and mental health chatbot with deep learning models and transformers.
* **[turkish-tweets-sentiment-analysis](https://github.com/bbchyo/turkish-tweets-sentiment-analysis):** Sentiment analysis project to determine tweet polarity in Turkish.
* **[tr-tweets-hate-speech](https://github.com/bbchyo/tr-tweets-hate-speech):** Detecting hate speech in Turkish tweets.
* **[mental-health-datasets](https://github.com/bbchyo/mental-health-datasets):** An evolving list of electronic media data sets used to model mental-health status.

### 🧬 Bioinformatics
* **[alphagenomeUI](https://github.com/bbchyo/alphagenomeUI):** AlphaGenome DNA Sequence Analysis UI - A BAGG AI & Google Collaboration.
